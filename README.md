🚀 Modern Developer Portfolio
A sleek, dark-themed, and fully responsive portfolio website built with Tailwind CSS and Alpine.js. This project showcases my journey as a Software Developer, featuring a dynamic project slider and a high-performance, minimalist UI.

#Key Features ✨

- Modern UI/UX: A dark-mode aesthetic with a "Glassmorphism" effect.
- Dynamic Project Slider: Interactive project showcase powered by Alpine.js.
- Fully Responsive: Optimized for all screen sizes from mobile to ultra-wide desktops.
- Performance Focused: Uses Tailwind CDN for rapid styling and smooth scroll animations.
- Technical DNA Section: A visual skill-bar representation of my core competencies.

#Tech Stack 🔩 

- Frontend: HTML5, Tailwind CSS
- Interactivity: Alpine.js
- Typography: Poppins (Google Fonts)
- Icons/Animations: Custom CSS Keyframes (Blink & Float)
